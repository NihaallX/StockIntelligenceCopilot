# Stock Intelligence Copilot - Language Style Guide

## Core Philosophy

We provide **balanced, confident, evidence-backed guidance** without creating urgency or commanding users. Think: "Professional analyst briefing" not "Infomercial sales pitch."

## Option B (Current Standard): Balanced & Defensible

**Characteristics:**
- Confident signal identification
- Evidence-based reasoning
- Relative language ("conditions suggest", "setup appears")
- No commands or time pressure
- Defensible if challenged

**Voice:** Professional analyst providing actionable intelligence

## Tone Calibration by Context

### High Confidence Signals (0.8+)
**Tone:** Confident but professional

**Example:**
```
STRONG BUY SIGNAL DETECTED

Technical setup indicates high-probability entry zone with favorable risk/reward profile. 
Multiple indicators align (RSI oversold + support bounce + volume surge). Entry: ‚Çπ2,500-2,550. 
Stop: ‚Çπ2,450 (2% risk). Target: ‚Çπ2,800 (10% upside).

Confidence: 87% | Risk: MEDIUM
```

### Medium Confidence Signals (0.6-0.8)
**Tone:** Measured, conditional

**Example:**
```
BUY SIGNAL DETECTED

Entry conditions appear favorable with appropriate position sizing. RSI showing oversold readings 
near established support. Setup suggests 2-3% portfolio allocation. Monitor for confirmation 
on increased volume.

Confidence: 72% | Risk: MEDIUM
```

### Low Confidence / Mixed Signals
**Tone:** Neutral, descriptive

**Example:**
```
SETUP NEUTRAL - Mixed Signals Present

Technical indicators show conflicting readings. RSI neutral at 45, price oscillating between 
support (‚Çπ2,500) and resistance (‚Çπ2,700). Probability-weighted scenarios suggest limited 
directional conviction. Monitoring recommended until clearer pattern emerges.

Confidence: 45% | Risk: MEDIUM
```

### Unfavorable Conditions
**Tone:** Cautionary, factual

**Example:**
```
CONDITIONS UNFAVORABLE - Risk/Reward Unattractive

Downside exposure (‚Çπ300) exceeds upside potential (‚Çπ150) at current levels. Multiple resistance 
overhead. RSI overbought at 78. Volatility elevated 40% above 30-day average. Setup does not 
meet minimum 2:1 reward/risk threshold.

Confidence: 81% (bearish) | Risk: HIGH
```

### Risk Mismatches (Profile Constraints)
**Tone:** Informative, protective

**Example:**
```
RISK ELEVATED - Volatility Exceeds Conservative Parameters

BUY signal present based on technical setup, but 30-day volatility (45%) exceeds conservative 
profile maximum (25%). Position sizing would need reduction to 1% (vs. typical 3%) to align 
with risk tolerance. Consider waiting for volatility compression.

Confidence: 76% | Risk: HIGH (for conservative profile)
```

## Headline Formulas

### "Today's Situations" Component
Format: `[Descriptive observation] + [neutral connector] + [outcome implication]`

**Examples:**
- ‚úÖ "Technical setup suggests potential upside for RELIANCE"
- ‚úÖ "Weakness signal detected as TCS tests support"
- ‚úÖ "Range-bound activity continues in ITC"
- ‚úÖ "Bullish momentum building in INFY above VWAP"

**Formula Breakdown:**
1. **Descriptive:** "Technical setup", "Weakness signal", "Range-bound activity"
2. **Connector:** "suggests", "detected", "continues"
3. **Outcome:** "potential upside", "tests support", "building above VWAP"

### Analysis Summaries (2-3 Sentences)
Format: `[Signal identification] + [Key evidence] + [Risk context]`

**Example:**
```
BUY signal identified with 78% confidence. Price bouncing off established support at ‚Çπ2,500 
with RSI oversold (28) and volume 2.3x average. Risk: MEDIUM with favorable 3:1 reward/risk ratio.
```

First mention of technical term in a session:
- ‚úÖ "RSI (Relative Strength Index) shows oversold at 28"
- ‚úÖ "VWAP (Volume Weighted Average Price) at ‚Çπ2,550"

## Quality Checklist

Before sending any user-facing text, verify:

- [ ] No command words (buy, sell, must, avoid)
- [ ] No urgency language (now, immediately, hurry)
- [ ] No guarantees (will, definitely, certain)
- [ ] Uses relative language (suggests, appears, indicates)
- [ ] Evidence-backed (cites RSI, VWAP, volume, etc.)
- [ ] Includes risk context (confidence %, risk level)
- [ ] Appropriate for confidence level (strong signal = confident, weak signal = measured)
- [ ] Follows Option B style (balanced, defensible)
